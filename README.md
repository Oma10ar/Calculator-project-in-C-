# C# Date & Time Utility Project

[cite_start]A specialized C# Console Application built on the **.NET Framework 4.7.2** that demonstrates efficient management of system-level settings and real-time date information using **Object-Oriented Programming (OOP)** principles[cite: 1, 3].

## 🚀 Overview
This project focuses on providing a clean, centralized way to access temporal data through a non-instantiable utility class. [cite_start]It serves as a foundational component for systems requiring time-based logic.

## 🛠 Technical Stack
* [cite_start]**Language:** C# [cite: 2, 3]
* [cite_start]**Framework:** .NET Framework 4.7.2 [cite: 1]
* [cite_start]**Project Type:** Console Application (Exe) [cite: 1]
* [cite_start]**Architecture:** Static Utility Pattern 

## ✨ Key Features
* [cite_start]**Static Configuration Management:** Implementation of a `Setting` class that prevents instantiation through a private constructor, ensuring a single point of truth.
* [cite_start]**Dynamic Date Retrieval:** Automatically fetches and formats:
    * [cite_start]**Day Number:** The numerical day of the current month.
    * [cite_start]**Day Name:** The full name of the current day of the week.
* [cite_start]**Clean Output:** Minimalist console interface for displaying system information.

## 📂 Project Structure
* [cite_start]**`Program.cs`**: Contains the core logic, including the `Setting` utility class and the main execution entry point.
* [cite_start]**`App.config`**: Manages runtime environment settings[cite: 1].
* [cite_start]**`Calculator Project.csproj`**: Defines project dependencies and build configurations[cite: 1, 2].

## ⚙️ How to Run
1. Ensure you have **Visual Studio 2017** or later installed.
2. [cite_start]Open the solution file `Calculator Project.slnx`[cite: 4].
3. Build and Run the application by pressing `F5`.
